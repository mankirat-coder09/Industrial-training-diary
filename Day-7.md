# 💻 Hands-on Project

## 📄 Basic RAG-Based Document Q&A System

### Project Description

Developed a simple **RAG pipeline-based Question Answering system** where user queries are answered using information retrieved from external documents instead of only relying on model memory.

---

### 🛠️ Technologies Used

- Python
- LLM (Ollama / OpenAI concept)
- Embeddings
- Vector Search (FAISS / Chroma concept)
- Document Loader
- Text Chunking

---

### 🔄 Project Workflow

1. Load Documents (PDF/Text)
2. Split text into chunks
3. Convert chunks into embeddings
4. Store embeddings in vector database
5. User enters query
6. Retrieve relevant chunks
7. Pass context + query to LLM
8. Generate final answer

---

### ✨ Features

- Document-based question answering
- Context-aware AI responses
- Faster and accurate information retrieval
- Reduces hallucinations in LLM outputs
