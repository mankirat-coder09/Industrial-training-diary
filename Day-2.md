To build a strong foundation in Python programming and understand the basics of Git and GitHub for version control and collaborative software development.

* Introduction to Python

* What is Git?

  * `git commit`
  * `git push`
  * `git pull`

* Successfully understood how to upload and manage projects on GitHub.

Day 2 strengthened my programming fundamentals by introducing Python and version control with Git and GitHub. These skills form the foundation for developing, managing, and collaborating on AI and software projects throughout the training.
